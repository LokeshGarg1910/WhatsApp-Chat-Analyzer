# WhatsApp Chat Analyzer

A WhatsApp Chat Analyzer capable of handling chat exports up to 200MB and converting unstructured text into structured datasets.

## Overview

* Extracts users, timestamps, message types, and metadata from exported WhatsApp chats.
* Provides analytical insights including activity timelines, busiest users, heatmaps, word clouds, and emoji statistics.
* Includes an interactive Streamlit dashboard supporting both individual and group-level insights.

## Technologies Used

* Python
* NumPy
* Matplotlib
* WordCloud
* Streamlit
